HTML Tags Overview

Basic HTML Tags

Tag Name Description
<html> HTML The root element that defines the entire HTML document. All other HTML elements are nested within this tag.
<head> Head Contains meta-information about the HTML document, such as the title and links to CSS or JavaScript files.
<body> Body Defines the content of the HTML document that is displayed in the browser (e.g., text, images, links).

Text Formatting Tags

Tag Name Description
<p> Paragraph Defines a paragraph of text. It is a block-level element that adds space before and after the text.
<strong> Strong Makes the enclosed text bold and adds semantic importance (used for important text).
<em> Emphasis Emphasizes text by italicizing it, usually indicating a change in tone or importance.

Link and Media Tags

Tag Name Description
<a> Anchor Defines a hyperlink, which links to another webpage or a specific section on the same page. The href attribute specifies the link destination.
<img> Image Embeds an image into the webpage. The src attribute specifies the image source, and alt provides alternative text for accessibility.
<audio> Audio Embeds audio content. The controls attribute can be added to show audio controls like play, pause, and volume.
